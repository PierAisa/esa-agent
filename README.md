# Progetto ESA Agent

Questo progetto contiene:
- un agente React in `/app/esa/page.js`
- una API serverless in `/app/api/esa/route.js`
- pronto per essere caricato su GitHub e Vercel

## ✅ Come usarlo
1. Crea un repository su GitHub
2. Copia dentro **tutti i file** di questo progetto
3. Vai su Vercel → *Add Project*
4. Seleziona il repo GitHub
5. Deploy automatico, funziona subito
